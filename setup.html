Configuração da API 🔑

Para carregar filmes e séries, o Heken precisa de uma chave da TMDB API — completamente gratuita e leva menos de 2 minutos para criar.

1
Acesse themoviedb.org/signup e crie uma conta gratuita
2
Vá em Configurações → API e clique em "Create" → escolha "Developer"
3
Copie a API Key (v3 auth) e cole abaixo

Já configurou antes? Ir direto para o catálogo
A chave é salva localmente no seu navegador.
